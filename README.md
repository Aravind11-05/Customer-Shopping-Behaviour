# Customer Shopping Behaviour Analysis

## Project Overview

This project analyzes customer shopping behavior using Python, SQL, and Power BI. The objective is to identify customer purchasing patterns, analyze sales trends, and generate business insights through data visualization.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* SQL
* Power BI

## Project Files

* customer_shopping_behavior.csv – Dataset used for analysis
* project.ipynb – Python analysis notebook
* sql.sql – SQL queries for data analysis
* Dash Board.png – Power BI dashboard screenshot

## Key Insights

* Customer purchase behavior analysis
* Product popularity analysis
* Sales trend analysis
* Data visualization using Power BI
